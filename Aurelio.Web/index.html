强大的功能特性

为现代工作流设计的全面功能集

多标签页工作流

创新的多标签页界面，支持标签页拖拽、分离到独立窗口、跨窗口移动，让您的工作流更加高效灵活

  • 标签页拖拽重排序
  • 分离标签页到新窗口
  • 跨窗口标签页移动
  • 智能标签页管理

强大的插件系统

模块化的插件架构，支持动态加载和卸载，轻松扩展应用功能

  • 动态插件加载
  • 标准化插件接口
  • 依赖关系管理
  • 独立配置系统

文件管理

内置强大的文件浏览器，支持多种文件格式预览和批量操作

  • 多格式文件预览
  • 批量文件操作
  • 强大搜索功能
  • 收藏夹管理

终端集成

内置终端模拟器，无需切换应用即可执行命令行操作

  • 内置终端模拟器
  • 多终端会话
  • 命令历史记录
  • 自定义主题

跨平台支持

基于 .NET 8 和 Avalonia UI，完美支持 Windows、macOS 和 Linux

  • Windows 7+
  • macOS 10.15+
  • Linux 发行版
  • 统一用户体验

现代化界面

基于 Fluent Design 的美观界面，支持深色/浅色主题切换

  • Fluent Design 风格
  • 深色/浅色主题
  • 自适应布局
  • 流畅动画效果
